I will describe the design I chose for the 3 categories (in the Homepage and the pages dedicated to it).

Gallery

As a big user of Instagram, (You can see his amazing work on Instagram), I decided to make a Gallery exactly like this one by keeping the same grid. For the Gallery page, I keept a simple page as he asked me for, with the 3 categories (Mariage, Portrait & Enterprise) displayed by a 3x3 grid.

Categories

In the Categories part, I put an image that represents the section in question, and for the description of the section, I just wrapped it on a camel color block to add this vintage touch that we talked about initially.

For the Categories page, I just make it a little bit fancier. I display all 3 categories as an unordered list and put 3 pictures of each category randomly on the screen with a half opacity. I had some interaction on hover. You can check that on Figma
